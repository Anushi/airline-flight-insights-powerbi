# ✈️ Airline Ticket Pricing & Flight Insights Dashboard

## 📌 Project Overview
This project presents an interactive **Airline Ticket Pricing & Flight Insights Dashboard** built using Microsoft Power BI. It analyzes airline ticket data to uncover insights into pricing patterns, flight distribution, and travel trends.

---

## 🎯 Objectives
- Analyze ticket pricing across different airlines and travel classes  
- Understand the relationship between distance and ticket price  
- Identify high-traffic origin and destination cities  
- Explore the impact of booking time on pricing  
- Build an interactive dashboard for data-driven insights  

---

## 📊 Dataset Description
The dataset includes the following fields:

- Ticket ID  
- Airline  
- Origin  
- Destination  
- Distance (km)  
- Class (Economy, Business, First)  
- Days Before Departure  
- Ticket Price (USD)  

---

## 📈 Key Features

### 🔹 KPI Metrics
- Total Flights  
- Total Revenue  
- Average Ticket Price  
- Average Distance  

---

### 🔹 Visualizations
- **Average Ticket Price by Airline**  
- **Distance & Revenue by Class**  
- **Flights by Origin**  
- **Flights by Destination**  
- **Flights by Class**  
- **Flight Routes Map**  

---

## 🎛️ Filters (Slicers)
- Airline  
- Class  
- Origin  
- Destination  

---

## 💡 Key Insights
- Ticket prices increase with travel class (Economy < Business < First)  
- Longer distances generally result in higher ticket prices  
- Certain airlines consistently charge higher fares  
- Major cities show higher flight activity  
- Booking timing influences ticket pricing trends
